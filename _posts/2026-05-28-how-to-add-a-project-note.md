Use a post when you want to explain something that does not fit cleanly inside the scrolling portfolio.

## Suggested Structure

- What triggered the note?
- What did you learn or change?
- What image helps prove the point?
- What would you do differently next time?

Keep it short. A blog post can be more reflective than a portfolio board, but it should still respect the reader's time.
